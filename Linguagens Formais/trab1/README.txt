# Guia de execução
Para executar o arquivo basta caminhar até o diretório em que o arquivo esta salvo --> cd /to/path

Dar permissão de execução para o arquivo
	chmod +x analisador_lexico.py

executar o arquivo
	python3 analisador_lexico.py
